# Wome Eats — Landing Page (Projet 1)

Landing page responsive pour le service de livraison de repas locaux Wome Eats,
réalisée dans le cadre du stage Full Stack chez DecodeLabs.

## Technologies utilisées

- HTML5 sémantique
- CSS3 (Grid + Flexbox, mobile-first)
- JavaScript vanilla

## Structure des fichiers

- `index.html` — structure sémantique de la page
- `reset.css` — reset CSS minimal
- `style.css` — layout, composants, media queries
- `script.js` — menu burger mobile
- `assets/images/` — images du projet

## Fonctionnalités

- Navigation responsive avec menu burger en mobile
- Section hero avec appel à l'action
- Grille de fonctionnalités (1 colonne → 2 → 3 selon la taille d'écran)
- Section "Comment ça marche" avec étapes numérotées
- Pied de page avec liens de contact cliquables

## Points de rupture (Breakpoints)

| Taille | Breakpoint |
|---|---|
| Mobile | Moins de 768px |
| Tablette | 768px et plus |
| Desktop | 1024px et plus |
